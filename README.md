# Professional Web Portfolio

This portfolio showcases selected web development projects completed during my undergraduate studies at Edge Hill University. It is designed to demonstrate my technical proficiency in frontend development, accessibility, and user experience design.

## Project Showcases
*   **Collections & Archives (C&A)**: My final year Research and Development project (CIS3425). A generous interface for exploring digital museum collections with AI-assisted exploration. (Utilizes data from a museum REST API).
*   **Animated Easter Card Creator**: A multi-step business utility demonstrating complex CSS animations and API integration.
*   **GymApp**: A data-driven workout tracker showcasing IndexedDB persistence and accessible UI design.

## Portfolio Key Features
*   Thematic Design: A "Rolling Yarn" visual concept that follows the user's progress, implemented via custom SVG path tracking.
*   Accessibility Mode: A dedicated "Easy Read" mode to assist users with motion sensitivity or visual impairments.
*   Horizontal Timeline: A structured, interactive representation of my development journey and skill acquisition.
*   Full Responsiveness: Optimized for all device classes from mobile to large desktop displays.

## Tech Stack
*   Languages: Vanilla JavaScript (ES6+), HTML5, CSS3 / SCSS.
*   Visuals: Scalable Vector Graphics (SVG) and complex keyframe animations.
*   Accessibility: Audited for WCAG 2.1 AAA compliance.
